# Print Pamphlets

Three print-ready pieces, built as self-contained HTML so anyone can turn them
into a PDF with no design software.

## How to make a PDF (any of them)

1. Open the `.html` file in **Google Chrome**.
2. **File → Print** (or ⌘P / Ctrl+P).
3. Destination: **Save as PDF**.
4. **Paper size: Letter**, **Margins: None**, **Background graphics: ON**.
5. For the brochure only: set **Layout: Landscape**.

Then send the PDF to any print shop (Staples, FedEx Office, a local Pasadena
printer) or print at home.

## The pieces

| File | What it is | Where to use it |
|---|---|---|
| `flyer.html` | One-page portrait flyer | Bulletin boards, coffee shops, coworking spaces, libraries, the Chamber — anywhere local people will see it. |
| `one-pager.html` | One-page sell sheet | A leave-behind to hand a business owner after a meeting, or attach as a PDF in an email. |
| `trifold-brochure.html` | 6-panel tri-fold brochure | Hand out at lunch-and-learns, events, and partner offices. Print **double-sided, flip on short edge, fold in thirds**. |
