# Marketing & Go-To-Market — Pasadena AI Institute

Everything you need to fill the founding cohort, in one place. Start with the
**Launch Playbook** — it tells you what to do, in order.

## Read in this order

1. **[LAUNCH-PLAYBOOK.md](./LAUNCH-PLAYBOOK.md)** — the operator's manual.
   Step-by-step from "today" to "first seat sold + first conference run":
   going live (domain, Stripe, Resend), booking a Pasadena venue, signing the
   first 10 seats, the Zoom sales-call script, and the run-of-show.
2. **[GO-TO-MARKET.md](./GO-TO-MARKET.md)** — the strategy. ICP & personas,
   positioning, objection rebuttals, the no-ads-first local channel plan, the
   30/60/90-day launch plan, unit economics ($995 × 24 seats), and KPIs.
3. **[BRAND.md](./BRAND.md)** — the source of truth for voice, colors, type,
   messaging pillars, and the canonical facts. Everything else follows this.

## Folders

- **[emails/](./emails/)** — 7 copy-paste email templates covering the whole
  funnel (cold outreach → interest follow-up → post-call → confirmation →
  prep → post-conference → waitlist nurture). See [emails/README.md](./emails/README.md).
  Fill in every `[bracketed]` placeholder before sending.
- **[pamphlets/](./pamphlets/)** — 3 print-ready pieces (flyer, one-pager sell
  sheet, tri-fold brochure) as HTML you can "Save as PDF." See
  [pamphlets/README.md](./pamphlets/README.md).

## The website

The funnel these materials drive to is the Next.js site in the repo root. It has
both conversion paths built in: an **interest form** (→ you follow up + book a
Zoom call) and **Stripe checkout** ($995). See the top-level
[`../README.md`](../README.md) to run and deploy it.
